Calcula la lletra del DNI

L’usuari/ària ha d’introduir un DNI i el programa ha de dir si la lletra és correcte
o no amb el nombre de DNI. Si la lletra no és correcta, ha de dir quina lletra és la
correcta.
Per calcular la lletra DNI has de prendre el número de DNI a verificar i dividir-lo entre 23.
La resta es reemplaça per una lletra que ja està assignada a aquest nombre mitjançant la taula adjunta:
Resto 0=>T // 1=>R // 2=>W // 3=>A // 4=>G // 5=>M // 6=>Y // 7=>F // 8=>P // 9=>D // 10=>X
11=>B // 12=>N // 13=>J // 14=>Z // 15=>S // 16=>Q // 17=>V // 18=>H // 19=>L // 20=>C
21=>K // 22=>E
